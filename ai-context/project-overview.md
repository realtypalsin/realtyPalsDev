# RealtyPals — Project Overview

## Vision

RealtyPals is an AI-powered real estate advisor designed to help Indian home buyers make better property decisions faster.

The product focuses on guidance, trust, and decision support rather than property discovery alone.

The AI advisor is the primary product experience.

Property data exists to support recommendations.

---

## Core Principles

### Conversation First

Users should be able to describe requirements naturally.

Examples:

* I need a 3 BHK under 1.5 crore near a metro.
* I want a family-friendly project in Noida.
* Show me builders with a strong delivery history.

Filters support the experience but should not be the primary interaction model.

---

### Trust First

Recommendations must be honest.

The system should:

* Explain tradeoffs
* Surface risks
* Highlight limitations
* Admit uncertainty

The platform must never hide property weaknesses.

---

### Decision Support

The goal is not maximizing views.

The goal is helping users confidently make purchase decisions.

## User Journey

1. User describes requirements.
2. AI gathers missing information.
3. AI recommends properties.
4. User evaluates options.
5. User compares projects.
6. User requests callback or visit.
7. Lead is handed to sales.
