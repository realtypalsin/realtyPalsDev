# Product: realtypals

## V1 Scope & Constraints [CRITICAL]
*   **Target Location:** Noida ONLY. Out-of-coverage areas must trigger a waitlist/honest messaging.
*   **Property Type:** New construction ONLY. No rentals, resale, or commercial properties.
*   **Platform:** Website only. Do not generate native mobile code.
*   **Core Identity:** AI advisor, not a generic listing portal. 

## Primary Workflows
1.  AI-driven property search and discovery (English/Hinglish).
2.  Decision tools (EMI, Stamp Duty, GST calculators).
3.  Entity and project verification (RERA compliance, Builder trust score).
4.  User Accounts (OTP / Google Login).
5.  Lead Handoff: WhatsApp integration and site visit scheduling.

## UI Constraints
*   Search input boxes are locked in design. Focus exclusively on search/query logic, data flow, and backend integration. Do not alter existing UI styling.
